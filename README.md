WiFi Deauthentication Tool

A lightweight and educational tool designed to demonstrate how WiFi deauthentication attacks work in real-world networks. This project is intended for cybersecurity learning, penetration testing, and network auditing in controlled environments only.

⚙️ Features
📶 Detect nearby WiFi networks (Access Points)
🔍 Monitor connected client devices
⚡ Simulate deauthentication packets
🧪 Useful for testing network resilience against DoS-style attacks
🖥️ Simple command-line interface for ease of use
🎯 Purpose

WiFi deauthentication attacks are commonly used in security testing to evaluate how vulnerable a wireless network is to disruption. This tool helps:

Understand wireless network behavior
Learn about 802.11 protocol vulnerabilities
Test security measures like WPA2/WPA3 protections
Demonstrate real-world attack scenarios in labs
